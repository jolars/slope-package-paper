# JSS Submission: SLOPE Package

This archive contains the submission materials for the Journal of Statistical Software.

## Directory Structure

- **paper/**: Complete source code and materials for the paper, including LaTeX files, images, code, and data needed to reproduce the results.

- **packages/**: Source code archives for the external SLOPE packages discussed in the paper:
  - libslope (C++)
  - sortedl1 (Python)
  - SLOPE (R)
  - SLOPE.jl (Julia)
